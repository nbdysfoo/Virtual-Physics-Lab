# virtual-physics-lab
This web application is designed to help students identify and address misconceptions in physics through a series of multiple-choice questions based on the Force Concept Inventory (FCI). The application provides a user-friendly interface for students to take quizzes and receive diagnostic feedback on their strengths and weaknesses.
